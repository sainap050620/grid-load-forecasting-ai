# Energy Grid Load Forecasting with Random Forest
This project implements a machine learning pipeline to predict electricity grid loads. 
It focuses on identifying patterns in consumption to help TSOs like TenneT manage grid stability.

### Key Features:
- **Data Engineering:** Pre-processing hourly time-series data.
- **Predictive Modeling:** Using Scikit-Learn to forecast next-day peak loads.
- **Insights:** Identifying correlation between temperature and grid stress.
